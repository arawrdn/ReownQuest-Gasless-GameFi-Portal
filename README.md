# ReownQuest: Gasless GameFi Portal (WalletConnect & Reown AppKit)

A full-stack GameFi portal showcasing the power of **Reown AppKit** and **WalletConnect**. This repository demonstrates how to deliver a truly seamless, gasless gaming experience by leveraging Smart Accounts and automated Smart Sessions.

### Key Features

* **Universal Onboarding:** Allow users to connect via **Email, Social Login**, or any of the **600+ supported wallets** (MetaMask, Trust, etc.).
* **Gasless Transactions:** Built-in **Gas Sponsorship** ensures players never need to hold native tokens for gas fees on every transaction.
* **Automated Sessions:** Use **Smart Sessions** to eliminate repetitive wallet signing for repeatable in-game actions like claiming daily rewards, leveling up, or crafting.
* **Multi-Chain Ready:** Easily manage assets and transactions across **EVM chains (Mainnet, Polygon, BSC)** and **Solana** via a single configuration.
* **Smart Accounts:** Utilizing Account Abstraction for enhanced security and simplified transaction logic.
